vue中使用scss

发表于 2020-07-27 分类于 vue ， 项目

scss语法

相关loader及作用

常用loader

npm i node-sass sass-loader css-loader style-loader -D

sass-loader 将 scss 转成 css 且依赖 node-sass

css-loader 解析 css语法插入到 js 中( webpack默认只能编译 js )

style-loader 将解析后 css 通过 <style></style> 添加到页面中

其他loader

npm i scss-resouce-loader -D

scss-resouce-loader 配置全局注册 scss or less 变量，依赖上面的loader (其他如 stylus 可用 style-resouce-loader)

webpack配置

  • test 匹配 scss 结尾的文件

  • use 使用到的loader; 执行顺序：从右向左

  • exclude 排除文件，即不处理 node_modules 下的 scss

  • scss-resouce-loader 参数含义：

    • patterns：字符串或数组，表示导入资源的路径，绝对路径

    • injector：‘prepend’ 或者 ‘append’， 表示资源导入的位置，在之前还是之后，样式后导入的会覆盖前面导入的

    • resolveUrl： 是否允许@import形式导入，默认true
